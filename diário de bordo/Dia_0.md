TL;DR: Vou criar um app para Samsung Galaxy Watch que toca música do meu servidor Plex. Detalhe: não sei a linguagem. O Claude vai escrever 95% do código.

Como falei ontem, estou lançando a braba vibe coder. Se os vendedores de ferramentas de IA dizem que a gente consegue fazer projetos com IA, vou tentar fazer um projeto completo com IA.

O projeto é simples cliente Plex para Wear OS:
- Login no servidor Plex
- Navegar por artistas/álbuns
- Tocar música direto no relógio
- Ouvir com fone de ouvido (não sei se é necessário alguma regra extra pra isso)
 
Como é um experimento, vou definir algumas regras:
- 95% do código será escrito pelo Claude
- Pesquisa zero fora do Claude Code e conversas com pessoas
- 95% da documentação também via Claude
- Sem revisão de código
- VSCode + Claude Code (plano Pro)

Vai ser um "build in public" de verdade. Diário de bordo de alguém que não sabe o que está fazendo, guiado por uma IA. Quero testar os limites. Até onde dá pra ir confiando cegamente em código gerado por IA? Funciona? Quebra? O app vai tocar música ou vai fritar meu relógio? 

O experimento vai durar até o app ficar pronto ou eu encher o saco. O que vier primeiro.

O link do repositório está disponível para quem pedir nos comentários.
